HTML Document Structure and Markup Basics

HTML: Hyper Text Markup Language

HTML Elements begins with a start tag, usually finishes with an end tag
They should be self-closed
Elements are either block-level or inline elements

Void/ Empty elements have no content and no end tag.
Do not add end tag to void/empty elements

White space characters like “spaces”, “tabs”, and “line breaks” do not affect the way content is presented
